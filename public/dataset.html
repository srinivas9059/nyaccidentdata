dataset

The dataset used in this project comprises the following columns:
  • Crash Date: Date of the accident.
  • Crash Time: Time of the accident.
  • Borough: Borough where the accident occurred.
  • Zip Code: Zip code of the accident location.
  • Latitude: Latitude of the accident location.
  • Longitude: Longitude of the accident location.
  • Location: Coordinates of the accident (Latitude, Longitude).
  • On Street Name: Street where the accident occurred.
  • Cross Street Name: Cross street at the accident location.
  • Off Street Name: Off-street location of the accident.
  • Number of Persons Injured: Count of individuals injured.
  • Number of Persons Killed: Count of individuals killed.
  • Number of Pedestrians Injured: Count of pedestrians injured.
  • Number of Pedestrians Killed: Count of pedestrians killed.
  • Number of Cyclist Injured: Count of cyclists injured.
  • Number of Cyclist Killed: Count of cyclists killed.
  • Number of Motorist Injured: Count of motorists injured.
  • Number of Motorist Killed: Count of motorists killed.
  • Contributing Factor Vehicle 1 to 5: Factors contributing to the accident from vehicles.
  • Collision ID: Unique identifier for the collision.
  • Vehicle Type Code 1 to 5: Type of vehicles involved in the accident.

Source

The Motor Vehicle Collisions crash table provides comprehensive details on crash events in New York City, with each row representing a specific incident. The data is sourced from police-reported motor vehicle collisions and is based on the mandatory completion of the MV104-AN police report form. This form is required for collisions involving injuries, fatalities, or property damage exceeding $1000. It's important to note that the data is preliminary and subject to change as MV-104AN forms are amended based on revised crash details. FORMS allows police officers to electronically enter all MV-104AN data fields using Department devices and stores this information in the crime data warehouse. This shift has enabled more detailed and comprehensive traffic safety analyses, aligning with the Citywide traffic safety initiative, Vision Zero, which aims to eliminate traffic fatalities. The implementation of FORMS represents a crucial step towards achieving Vision Zero's goals by facilitating in-depth analyses of traffic collision data.

Patterns and Key Points

Evolution of Data Collection Systems:

The NYPD's approach to collecting data on motor vehicle collisions has evolved over time.

Initially, there was no uniform data collection procedure for traffic safety across precincts.

Focus on Traffic Fatalities:

The overarching goal is to address and reduce traffic fatalities, aligning with the Vision Zero initiative.

The emphasis on detailed data collection and analysis is driven by a commitment to improving traffic safety outcomes.

Ongoing Revision of Data:

The acknowledgment that the data is preliminary and subject to change highlights an ongoing commitment to accuracy.

MV-104AN forms are amended based on revised crash details, ensuring the most up-to-date information.
